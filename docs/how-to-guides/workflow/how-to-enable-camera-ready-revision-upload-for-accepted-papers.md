# How to enable Camera Ready Revision Upload for accepted papers

From the [venue request form](../../getting-started/hosting-a-venue-on-openreview/navigating-your-venue-pages.md#venue-request-form), click the ‘Submission Revision Stage’ button to set up camera-ready revisions for papers. To view all camera-ready versions submitted to your venue, refer to the instructions [here](../data-retrieval-and-modification/how-to-get-all-notes-for-submissions-reviews-rebuttals-etc.md).&#x20;
