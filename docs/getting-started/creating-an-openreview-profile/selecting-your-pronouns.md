# Selecting your pronouns

The Profile Edit page will allow users to select pronouns for their profile. This dropdown field is optional and it contains four pre-defined options to choose from: they/them, he/him, she/her, and not specified.

<figure><img src="../../.gitbook/assets/Screenshot 2025-12-12 at 3.31.54 PM.png" alt=""><figcaption></figcaption></figure>

To customize your pronouns, type them into the textbox and click on your custom value in the dropdown to select it.

<figure><img src="../../.gitbook/assets/Screenshot 2025-12-12 at 3.33.11 PM.png" alt=""><figcaption></figcaption></figure>



Note: To protect user privacy, pronoun data cannot be retrieved in the API response.
